Наведите порядок в админ-меню WordPress: скрывайте лишние пункты или собирайте их в один хаб с вкладками — отдельно для админа и для персонала. Без правки чужих плагинов.

== Description ==

**WP Admin Menu Hub** превращает раздутый админ-сайдбар в чистое, понятное меню за пару кликов — не трогая код плагинов.

Когда установлено много плагинов (доставка, синхронизация, аналитика, SMTP…), каждый вешает свой пункт в меню, и сотруднику видно всё подряд. Этот плагин даёт оставить только нужное, а остальное **скрыть** или **встроить вкладкой** в единый хаб.

= Возможности =

* **3 действия на любой пункт меню:** Показать · Скрыть · Встроить (вкладкой в хаб).
* Работает с **любыми** пунктами — плагинов и ядра WP (Записи, Плагины, Внешний вид, Инструменты…), верхнего уровня и подпунктов.
* **Применение по ролям** — правила можно включить только для не-администраторов: владелец видит полное меню, персонал только рабочее.
* Настраиваемое название и иконка хаба.
* Один хаб-пункт + карточки/вкладки поверх встроенных страниц.
* Ничего не ломает: страницы остаются доступны, их JS работает.
* Полностью обратимо: деактивация возвращает меню как было.

= Чем отличается =

Большинство «прячущих» решений используют `remove_menu_page()` — но это **ломает доступ к странице** (WordPress начинает отвечать «недостаточно прав»). WP Admin Menu Hub прячет пункты через **CSS**, поэтому страница остаётся полностью рабочей и доступной по URL.

== Installation ==

1. Загрузите папку `wp-hub` в `/wp-content/plugins/`.
2. Активируйте плагин в разделе «Плагины».
3. Откройте **Настройки → Menu Hub** и настройте.

== Frequently Asked Questions ==

= Скрытые пункты пропадают навсегда? =
Нет. Они только прячутся из меню — страницы доступны по прямой ссылке и возвращаются, как только снимете «Скрыть». Деактивация плагина возвращает всё.

= Сломает ли это страницы настроек других плагинов? =
Нет. Пункты не удаляются и страницы не переносятся — поэтому их скрипты (кнопки «проверить связь» и т.п.) работают как обычно.

= Можно скрыть меню только для сотрудников, а себе оставить? =
Да. В настройках есть режим «Всем, кроме администраторов».

= Нужен ли WooCommerce? =
Нет. Плагин работает с любым админ-меню WordPress.
